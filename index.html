Beautiful, real-time chat in O(1) lines of code:

          /* Client-side */
import ChatClient from "react-chat-client";
ChatClient.initialize();

/* Server-side */
const ChatServer = require("react-chat-server");
const chat = new ChatServer(server);
app.use(chat.expressMiddleware());

Installation

          npm install react-chat-client react-chat-server --save
